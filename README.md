# java-racingcar

## 기능 목록 📝

### 기능 

- [x] 자동차 이름 쉼표로 구분
- [ ] 0에서 9사이의 랜덤값 구하기
  - [ ] 랜덤값이 4이상이면 전진
  - [ ] 랜덤값이 3이하면 정지
- [ ] 게임 완료 후 우승자 정하기

### 입력

- [x] 경주할 자동차 이름 입력
- [x] 시도할 횟수 입력

### 출력

- [x] 안내 문구 출력
- [ ] 실행 결과 출력
- [ ] 최종 우승자 출력


### 예외

- [ ] 자동차 이름 구분자가 쉼표가 아닌 경우
- [x] 이름 길이가 최소 1자, 최대 5자가 아닌 경우
- [x] 자동차 수가 2대 미만인 경우
